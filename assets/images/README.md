# Menu Image

## Instructions

Place your menu image here as `menu.jpg` (the pug with glasses image).

### Quick Steps:

1. Save the pug image you want to use as: `menu.jpg`
2. Place it in this directory: `assets/images/`
3. Restart the bot

The bot will automatically display this image when users:
- Send `/start` command
- Click "Back to Menu" button
- Cancel an operation

### Image Requirements:

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended max 5MB
- **Dimensions**: Recommended 800x600px or similar
- **Name**: Must be exactly `menu.jpg`

### Fallback:

If no image is found, the bot will display text-only menu (backwards compatible).
